256kb.cloud

Tiny baby cloud provider. All you get is:

  • One index.html page up to 256kb in size. No external links allowed.
  • One server-side WebAssembly binary with only 256kb of memory.
  • Server code is submitted and compiled to WebAssembly. Compilers for Go, Zig, and Rust are available.
  • Everything is immutable. Edits to existing applications will create a new application.
  • Everything is open, all code, logs, and runtime information are available.
